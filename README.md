## Cara Menjalankan

1. Clone atau Download repositori ini ke komputer Anda.

```sh
git clone https://github.com/mby444/rsa-from-scratch.git
```

2. Buka terminal atau command prompt di direktori proyek.

3. Jalankan program dengan perintah berikut:

```sh
python main.py
```

4. Masukkan pesan teks saat diminta, lalu tekan Enter.
